Plan de estudios Ofimática

Este es el siguiente manejo de los modulos para la carrera Técnico Ofimática

Modulo I

Gestiona hardware y software de la ofimática

  • Submódulo 1 - Instala y configura equipo de cómputo y periféricos
  • Submódulo 2 - Instala y configura sistemas operativos y aplicaciones de la ofimática
  • Submódulo 3 - Gestiona archivos y dispositivos ofimáticos

Modulo II

Gestiona información de manera local

  • Submódulo 1 - Gestiona información mediante el uso de procesadores de texto
  • Submódulo 2 - Gestiona información mediante el uso de hojas de cálculo
  • Submódulo 3 - Gestiona información mediante el uso de software de presentaciones

Modulo III

Gestiona información de manera remota

  • Submódulo 1 - Gestiona información mediante el uso de software en línea
  • Submódulo 2 - Gestiona recursos mediante el uso de redes de computadoras

Modulo IV

Diseña y gestiona bases de datos ofimáticas

  • Submódulo 1 - Diseña bases de datos ofimáticas
  • Submódulo 2 - Gestiona información mediante el uso de sistemas manejadores de bases de datos ofimáticas

Modulo V

Establece comunicación ofimática

  • Submódulo 1 - Gestiona información a través de plataformas digitales
  • Submódulo 2 - Establece comunicación y gestiona información mediante el uso de dispositivos móviles
